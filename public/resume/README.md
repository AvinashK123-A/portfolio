# Resume Folder

Place the file `Avinash_Reddy_Resume.pdf` in this folder.

The Download Resume button on the portfolio links to:
`/resume/Avinash_Reddy_Resume.pdf`

## Steps to add your resume PDF:
1. Export your resume as PDF named `Avinash_Reddy_Resume.pdf`
2. Upload it to this folder: `public/resume/Avinash_Reddy_Resume.pdf`
3. The Download Resume button will automatically serve the file

## Download button behavior:
- Button uses HTML5 `download` attribute for direct browser download
- File is served from Next.js `public` folder (static assets)
- No external links — file is hosted within the portfolio itself
